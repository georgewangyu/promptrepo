# PromptRepo

A curated collection of thinking frameworks and prompts, transformed into actionable AI instructions.

## The Idea

Great thinkers like Ray Dalio, Charlie Munger, and others have developed powerful mental models for decision-making, problem-solving, and clarity. This repository takes those frameworks and converts them into structured, ready-to-use AI prompts.

**Why?** Because having Bridgewater's culture of radical transparency or Munger's lattice of mental models "in your pocket" as AI prompts creates a brutally honest thinking partner available on demand.

## Structure

```
promptrepo/
├── AGENTS.md              # Bootstrap for AI assistants
├── SOUL.md                # Identity and style guide
├── README.md              # This file
├── decision-making/       # Prompts for better decisions
├── problem-solving/       # Mental models and analysis frameworks
├── creativity/            # Lateral thinking and ideation
├── communication/         # Feedback, negotiation, clarity
└── planning/              # Goal setting and strategic thinking
```

## Quick Start

1. **Browse by use case** — Find the folder that matches what you're working on
2. **Pick a framework** — Each framework has ready-to-use prompts
3. **Copy and customize** — Paste into your AI tool, fill in your context

## Example: Ray Dalio's Radical Transparency

```
"If we're radically transparent, what's really true about [situation]?"

AI gives you the pattern you've been avoiding, not the story you've been telling yourself.
```

See [`decision-making/ray-dalio-principles/`](decision-making/ray-dalio-principles/) for the full set.

## Contributing

This is a living collection. Frameworks should be:
- **Actionable** — Prompts you can use immediately
- **Sourced** — Attribute the original thinker/framework
- **Explained** — Include why it works and when to use it
