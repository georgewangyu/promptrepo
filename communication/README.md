## Core Concepts

### Radical Transparency
Say what you actually think, with the intent to help, not harm.

### Nonviolent Communication
Observe → Feel → Need → Request

### Steel Manning
Present the strongest version of the other person's argument before responding.

### SBI Model
Situation-Behavior-Impact for clear feedback.

## When to Use These

- Difficult conversations
- Giving or receiving feedback
- Resolving misunderstandings
- When emotions are running high
- Negotiations

## Quick Prompt

**For difficult conversations:**

```
I need to discuss [sensitive topic] with [person].

Help me prepare:
1. What's my actual goal in this conversation?
2. What might they be feeling/need that I'm not seeing?
3. What's the steel man version of their perspective?
4. How can I be radically transparent but constructive?
5. What specific behavior and impact can I reference (SBI)?
```
