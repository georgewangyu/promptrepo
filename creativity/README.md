## Core Concepts

### Constraint as Catalyst
Limitations often produce the most creative solutions.

### Analogical Thinking
Solve this problem by finding a domain that solved something similar.

### Random Entry
Use random stimuli to break fixed thinking patterns.

### SCAMPER
Substitute, Combine, Adapt, Modify, Put to other uses, Eliminate, Reverse.

## When to Use These

- Brainstorming sessions
- Creative blocks
- Product/feature ideation
- Content creation
- When stuck in conventional thinking

## Quick Prompt

**For creative blocks:**

```
I need ideas for [project/challenge]. I'm feeling stuck in conventional thinking.

Use analogical thinking:
1. What domains have solved similar structural problems?
2. What would [unrelated field] do with this challenge?
3. What constraints could I add to force creative solutions?
4. Generate 10 wild ideas first, then refine.
```
