# Prompts

## Full Decision-Making Prompt

```text
I need help making a decision about [decision].

Apply a Ray Dalio-style principles lens:

1. Radical transparency
- What is really true here that I may be avoiding?
- What are the uncomfortable facts I should state plainly?

2. Believability weighting
- Whose opinion matters most on this issue based on actual track record?
- Which views sound confident but are not especially believable in this domain?

3. Second-order consequences
- What happens immediately if I choose each option?
- What happens after that?
- What are the likely second- and third-order effects?

4. The machine
- What system or cause-effect loop is driving this situation?
- Which recurring pattern is producing the same outcome?

5. Principles
- What principle should govern this decision?
- If I had to write one rule from this decision for future me, what would it be?

Return:
- the real situation
- the strongest arguments for each option
- the believable perspective
- the likely second-order consequences
- the decision principle I should use
```

## Compact Version

```text
Use a Ray Dalio decision lens:
- Tell me what is really true, not what is comforting.
- Weight advice by actual track record in this domain.
- Show the second- and third-order consequences.
- Identify the underlying machine or pattern.
- End with the principle that should govern the decision.
```

## Pain + Reflection Prompt

```text
I just experienced a painful outcome in [situation].

Help me use Ray Dalio's "pain + reflection = progress" idea:
1. What exactly happened?
2. What part was painful because reality contradicted my expectations?
3. What assumption or principle failed?
4. What pattern does this reveal?
5. What explicit principle should I adopt so I handle this better next time?
```
